A simple, interactive Ping Pong scorekeeping app built with HTML, CSS, and JavaScript. This app allows players to keep track of scores, select the winning score, and reset the game. It also functions as a Progressive Web App (PWA) with offline capabilities.

---

## Features

- **Interactive Scoring**: Add points for Player One and Player Two with buttons.
- **Winning Score Selection**: Choose the winning score from a dropdown menu.
- **Game Reset**: Reset the game at any time.
- **PWA**: Installable on mobile devices and works offline.
